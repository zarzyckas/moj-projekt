# 🤝 Wkład w projekt

Dziękujemy za zainteresowanie! Aby wnieść wkład:

1. Forkuj repozytorium
2. Utwórz nową gałąź (`git checkout -b feature/nazwa`)
3. Wprowadź zmiany
4. Zatwierdź (`git commit -m 'Opis zmiany'`)
5. Wypchnij gałąź (`git push origin feature/nazwa`)
6. Utwórz Pull Request

git add CONTRIBUTING.md
git commit -m "Dodano plik CONTRIBUTING.md"
git push
